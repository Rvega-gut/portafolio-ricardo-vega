# 🐍 Python

Proyectos desarrollados con Python, enfocados en el análisis, limpieza y modelamiento predictivo de datos.

## Proyecto destacado

**Predicción de reinserción hospitalaria en pacientes con diabetes**  
Análisis exploratorio y modelo predictivo para estimar la probabilidad de reingreso hospitalario en pacientes diabéticos.
Se trabajó con el dataset *Diabetes 130-US Hospitals* (UCI Machine Learning Repository), aplicando limpieza de datos,
transformación y descategorización de variables, selección de variables mediante pruebas estadísticas (chi-cuadrado y Spearman),
y comparación de modelos de clasificación (Random Forest, Regresión Logística, Naive Bayes).
